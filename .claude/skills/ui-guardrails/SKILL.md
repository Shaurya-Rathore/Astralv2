---
name: ui-guardrails
description: Preserve and improve premium dashboard UI without accidental redesigns.
---

Apply these rules whenever editing frontend UI:

- keep current layout, hierarchy, and styling unless explicitly asked otherwise
- improve polish by refining spacing, typography, labels, and empty states
- do not replace polished components with simpler ones
- keep filters, chips, cards, and tables visually coherent
- avoid introducing generic admin-panel styling
- preserve dark/light mode if present
- preserve shadcn-style composition if present
- no placeholder text
- no dead buttons
